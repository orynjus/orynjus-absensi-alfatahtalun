# orynjus-absensi-alfatahtalun

Sistem Absensi Digital untuk MTs Al Fatah Talun

## 🚀 Quick Start

### Development
```bash
npm install
npm run dev
```

### Production Deployment
Lihat panduan deployment di:
- [DEPLOYMENT_GUIDE.md](./DEPLOYMENT_GUIDE.md)
- [FREE_HOSTING_GUIDE.md](./FREE_HOSTING_GUIDE.md)
- [RAILWAY_DEPLOY.md](./RAILWAY_DEPLOY.md)

## 📋 Fitur
- QR Code absensi
- Import data dari Google Sheets
- Notifikasi WhatsApp
- Multi-role (Siswa, Guru, Wali Kelas, Admin)
- Dashboard analytics
- Export data ke CSV

## 🛠️ Tech Stack
- Node.js + Express
- PostgreSQL (Neon Cloud)
- React + Vite
- TailwindCSS
- Drizzle ORM

## 🚀 Deployment

### Railway (Recommended)
```bash
# Repository: https://github.com/orynjus/orynjus-absensi-alfatahtalun
# Branch: main
# Build: npm run build
# Start: npm start
```
